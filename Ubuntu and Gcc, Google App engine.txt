1  Download Ubuntu ISO from the official Ubuntu website.
2  Open VirtualBox and click New.
3  Name your virtual machine, choose a location, and select the OS type as Ubuntu.
4  Assign RAM (at least 2048 MB recommended).
5  Create a virtual hard disk: choose VDI format and Dynamically Allocated storage.
6  Set disk size (20 GB or more recommended) and confirm.
7  Start the virtual machine and when prompted, select the downloaded Ubuntu ISO file.
8  Choose Install Ubuntu.
9  Select keyboard layout and click Continue.
10 Choose installation type (Normal is recommended) and click Install Now.
11 Confirm disk changes (this affects only the virtual disk).
12 Select your time zone, set your username and password.
13 Wait for installation to complete and reboot the VM.

int main() { 
float numl, num2, sum; 
printf("Enter the first number: "); 
scanf("%f", &numl); 
printf("Enter the second number: "); 
scanf("%f", &num2);
sum = numl + num2; 
printf("The sum of %.2f and %.2f is %.2f\n", numl, num2, sum); 
return 0; 
)
